## neofs-cli bearer print

Print binary-marshalled bearer tokens from file or STDIN in JSON format

### Synopsis

neofs-cli bearer print [FILE]
With no FILE, or when FILE is -, read standard input.

```
neofs-cli bearer print [flags]
```
